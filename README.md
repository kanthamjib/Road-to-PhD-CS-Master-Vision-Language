# Road-to-PhD-CS-Master-Vision-Language
[Road to PhD-CS Course: Master Vision Language!]

This is a project, builded for education proposes. In this project, we will dive into the NLP and CV field by learning, deriving mathematical concepts, and concept implementation by coding from scratch.

There're 3 main cores of this course, and 20 chapters in overall. Following:

📥 Phase 1: NLP Foundations
Chapter 1.1 Tokenization and Embeddings
Chapter 1.2 Word2Vec
Chapter 1.3 RNN
Chapter 1.4 LSTM/GRU
Chapter 1.5 Seq2Seq
Chapter 1.6 Attention
Chapter 1.7 Transformer
Chapter 1.8 autoregressive LM (GPT)
Chapter 1.9 (Project 1 NLP) Building mini LLM

📥 Phase 2: CV Foundations
Chapter 2.1 CNN basics
Chapter 2.2 LeNet/AlexNet
Chapter 2.3 ResNet
Chapter 2.4 Feature maps + GradCAM
Chapter 2.5 ViT
Chapter 2.6 Contrastive vision learning
Chapter 2.7 (Project 2 CV) Building mini Object Detection Model

📥 Phase 3: Multimodal Foundations
Chapter 3.1 Image-text embeddings
Chapter 3.2 CLIP
Chapter 3.3 Cross-attention
Chapter 3.4 (Final Project) Building mini VLM

*Objective Goals*
1. Understand and implement core NLP, CV, and multimodal architectures *from scratch* with minimal reliance on high-level libraries.

2. Can explain (Every Questions of it!):
* what the technique is?,
* what problem it was designed to solve?,
* why it was historically important?,
* and how it works mathematically (must and can derive) and architecturally?.

3. Analyze the strengths, limitations, and trade-offs of each method through implementation and experimentation.

4. .ipynb will be used for notebook implementation and .py scripts in phase*src/ used for algorithms source code implementation
